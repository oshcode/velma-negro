# Assets Folder

This folder contains all game assets organized by type.

## Folder Structure

- **characters/** - Cat and dog sprites (running, jumping animations)
- **items/food/** - Collectible food items (apples, fish, bones, etc.)
- **items/obstacles/** - Hazards to avoid (spikes, bad food, etc.)
- **backgrounds/** - Ground tiles and background layers
- **ui/** - User interface elements (buttons, score display, etc.)

## Next Steps

1. Read `QUICK_START_ASSETS.md` in the project root for download instructions
2. Download asset packs from Kenney.nl or Itch.io
3. Extract and organize files into these folders
4. Start building your game!

## Recommended Asset Sizes

- Characters: 32x32 or 64x64 pixels
- Food items: 16x16 or 32x32 pixels
- Obstacles: 32x32 pixels
- UI elements: Varies (buttons can be larger)

Keep all assets at similar resolutions for consistency!
